# Opinion maximization on social trust networks based on game theory and DQN method

## 摘要

社交网络中的观点传播会显著影响舆情、营销和社会治理。观点最大化（Opinion Maximization, OM）问题的目标，是选择合适的种子节点，使整个网络的总体观点尽可能正向。现有研究大多关注“影响了多少节点”，但对网络中信息接受与融合机制的讨论仍然不足。因此，本文基于扩展的连续观点动力学模型，研究社会信任网络（STN）上的观点最大化问题。首先，作者把个体的“固执程度”视为一种策略选择，并据此构建了基于博弈论的连续观点演化模型。随后，提出一种基于深度强化学习的两阶段方法 T-DQN：第一阶段使用启发式规则筛选候选种子，第二阶段使用深度 Q 网络确定最终种子，并结合优先经验回放与软更新机制提升学习效率。实验表明，T-DQN 在最大化最终观点总和方面优于多种基线算法，并且对不同网络结构和不同初始观点设置（随机选择/回音室）都具有较强鲁棒性。此外，关于初始合作比例和网络结构的仿真实验，也为舆论引导与信息传播优化提供了新的启发。

## 1. 引言

随着社交媒体的快速发展，社交网络对个体的影响越来越明显，人际关系也更加复杂多样。传统社会网络研究经常采用无向或无权图，但现实中用户之间的影响强度并不一致，而且关系还可能包含信任/不信任、喜欢/反感等带符号属性。因此，使用有向、带权、带符号的社会信任网络更符合实际。

影响力最大化（IM）是社交网络分析中的经典问题，目标是在网络中选择一组种子节点，使信息扩散后的激活节点数最大。随着研究发展，竞争性影响力最大化和观点最大化问题逐渐受到关注。观点最大化与传统 IM 的不同之处在于，它不只考虑是否被影响，还考虑用户观点值会随时间连续演化，因此更接近现实。

OM 的核心包括两部分：观点动力学模型与种子节点选择算法。就观点动力学而言，本文关注连续观点演化，并指出“固执”不应被简单视为个体固定属性，而应视为随环境变化的策略选择。基于经典 Friedkin-Johnsen 模型，本文引入博弈论，把个体是否愿意更新观点建模为合作/背叛策略选择，并用局部收益刻画观点在周边环境中的一致性优势。

在种子节点选择方面，贪心算法通常效果较好但代价高，启发式算法效率高但精度有限。深度学习和强化学习方法开始用于种子选择，其中 DQN 比传统 Q-Learning 更适合处理连续或高维状态空间。为此，本文提出面向社会信任网络的两阶段 DQN 方法，用于实现观点最大化。

本文贡献主要有三点：

1. 从策略选择角度提出新的观点传播模型，把固执性建模为演化过程中的动态决策。
2. 提出适用于连续状态空间的两阶段观点最大化算法 T-DQN，通过把动作空间限制为 5 种候选策略降低复杂度。
3. 在不同网络与不同初始观点设置下验证算法有效性与鲁棒性，并进一步分析初始合作比例和网络结构对最终观点的影响。

## 2. 观点最大化框架

### 2.1 问题定义

社会信任网络表示为有向带权带符号图 `G=(V,E,W)`。其中：

- `V` 为节点集合；
- `E` 为边集合；
- `W={w_ij}` 为边权矩阵，`w_ij in [-1,1]`；
- `w_ij > 0` 表示 `i` 对 `j` 的信任；
- `w_ij < 0` 表示 `i` 对 `j` 的不信任；
- `w_ij = 0` 表示不存在从 `i` 到 `j` 的边。

节点观点 `o in [-1,1]`：

- `o in (0,1]` 表示正向观点；
- `o in [-1,0)` 表示负向观点；
- `o = 0` 表示中立。

种子节点集合 `S` 的作用是传播正向观点。论文中把种子节点观点固定为 `+1`，同时把其策略固定为合作，以保证其持续向邻居传播正向观点和合作策略。

观点最大化目标是在给定预算 `k` 和传播时长 `T` 时，选择 `k` 个种子节点，使最终总体观点

`Phi(S) = sum_{i in V} o_i^T`

最大。

### 2.2 基于博弈论的观点动力学模型

作者把个体固执性视为策略选择。每个节点在时刻 `t` 具有：

- 连续观点 `o_i^t in [-1,1]`
- 二元策略 `z_i^t in {0,1}`

其中：

- `z_i^t = 1` 表示合作，愿意参考邻居更新观点；
- `z_i^t = 0` 表示背叛，坚持原观点不变。

观点更新规则可以理解为：

- 若节点选择背叛，则保持原观点；
- 若节点选择合作，则将自身观点与邻居观点按照有向权重和邻居局部收益进行融合。

论文中的更新式可写成：

`o_i^{t+1} = (1-z_i^t)o_i^t + z_i^t * (o_i^t + sum_{j in N_i^out} w_ij B_j^t o_j^t) / (1 + sum_{j in N_i^out} |w_ij| B_j^t)`

这里 `B_j^t` 表示邻居 `j` 在局部网络中的加权平均收益，表示其观点在局部环境中的一致性优势。节点更愿意参考那些“局部收益更高”的邻居。

为了度量局部一致性，作者定义节点 `i` 面对邻居 `j` 时的成对局部收益 `l_ij^t`：

- 若 `w_ij > 0`（信任），观点越接近，收益越高：
  `l_ij^t = 1 - 0.5 * |o_j^t - o_i^t|`
- 若 `w_ij < 0`（不信任），观点越不同，收益越高：
  `l_ij^t = 0.5 * |o_j^t - o_i^t|`

节点 `i` 的加权平均收益为：

`B_i^t = sum_{j in N_i^out} |w_ij| l_ij^t / sum_{j in N_i^out} |w_ij|`

策略更新方面，节点只模仿其“信任的出邻居”。先按权重概率选择一个被信任邻居 `j` 作为比较对象，再依据 Fermi 规则以概率

`W^{t+1}(z_i -> z_j) = 1 / (1 + exp(-beta(B_j^t - B_i^t)))`

采用邻居 `j` 的策略，其中 `beta` 表示选择强度。论文默认 `beta=10`，表示中等偏强选择。

网络中的合作比例定义为：

`P_coop = (sum_i z_i) / N`

初始合作比例记为 `P_coop^0`。

### 2.3 基于 DQN 的两阶段观点最大化

由于 OM 问题中的状态是连续的，而且目标函数不满足次模性质，本文使用 DQN 代替传统贪心或表格型 Q-Learning。

#### 2.3.1 候选种子筛选

第一阶段使用启发式规则筛选候选种子。节点 `i` 的影响力由三部分组成：

1. 自身基础影响力；
2. 来自出邻居的影响；
3. 作用于入邻居的影响。

公式为：

`p_i = B_i^0 + sum_{v in N_i^out} B_v^0 w_iv + sum_{u in N_i^in} B_i^0 w_ui`

随后按 `p_i` 从高到低排序，取前 `xi * k` 个节点作为候选种子集合 `C`。论文实验中 `xi=2`。

#### 2.3.2 最终种子选择

第二阶段使用 DQN 从候选集合中逐个选出最终种子。强化学习元素定义如下。

**奖励（Reward）**

加入一个新种子后，最终总体观点的增量：

`r_m = Phi(S U {v*}) - Phi(S)`

**动作（Action）**

不是“直接选某个节点”，而是从 5 种选种策略中选一种，从而显著压缩动作空间：

1. `MaxDegree`：选出度最大的候选节点；
2. `Blocking`：选入度最大的候选节点；
3. `MixStrategy`：按入度和出度的加权和排序，文中取 0.5:0.5；
4. `CbC`：基于社区结构的中心性；
5. `CI`：Collective Influence，半径取 2。

**状态（State）**

使用与网络规模无关的 25 维连续向量：

1. 合作比例 `P_coop`；
2. 全网平均观点；
3. 当前已选种子统计：已选数量、平均度、最大度；
4. 出度分布直方图 10 维；
5. 入度分布直方图 10 维。

**DQN 结构与训练**

- 策略网络和目标网络均为两层隐藏层的全连接网络；
- 每层 256 个神经元；
- 激活函数为 ReLU；
- 初始探索率 `epsilon=0.4`；
- 每次决策后做 `epsilon=max(0.9*epsilon, 0.1)`；
- 经验回放池大小 `10000`；
- batch size `32`。

论文引入优先经验回放（PER）。新经验的初始优先级设为 1；已有经验的优先级与 TD 误差绝对值相关：

`priority_n = (|delta_n| + eps)^alpha`

其中 `eps=1e-6`，`alpha=0.6`。

目标 Q 值：

`y_n = r_n + gamma * max_{a'} Q_target(s'_n, a')`

TD 误差：

`delta_n = y_n - Q_policy(s_n, a_n)`

重要性采样权重按论文进行修正并归一化，损失函数是加权均方误差。目标网络采用软更新：

`theta_target <- tau * theta_policy + (1-tau) * theta_target`

其中 `tau=0.05`。

最终的 T-DQN 流程是：

1. 用启发式规则生成 `xi*k` 个候选种子；
2. 初始化 DQN、经验回放池与社区结构；
3. 重复 `k` 次：
   - 抽取当前状态；
   - 用 `epsilon`-greedy 选一个动作策略；
   - 按该策略从候选集合里挑出一个节点；
   - 计算奖励并加入种子集合；
   - 存入经验并更新策略网络；
   - 周期性软更新目标网络。

## 3. 数据与验证

### 3.1 数据集与参数设置

论文在 6 个网络上验证算法：

1. **BBV**：5000 节点，30000 边，平均度 12，正边占比 80%
2. **SBM**：5000 节点，16954 边，平均度 6.78，正边占比 79.80%
3. **WS**：5000 节点，30000 边，平均度 12，正边占比 80%
4. **WikiElec**：7118 节点，103747 边，平均度 29.15，正边占比 78.79%
5. **Gnutella**：22687 节点，54705 边，平均度 4.82，正边占比 80%
6. **Reddit**：54075 节点，234792 边，平均度 8.68，正边占比 92.14%

对于无符号网络，作者将约 80% 的边设为正边，以与 WikiElec 的正边比例接近。实验中节点初始观点从 `[-1,1]` 随机采样，初始策略在合作/背叛间等概率随机选择。所有节点同步更新观点，再同步更新策略。为降低随机性，每组实验重复 20 次并报告平均值。

边权设置方面，保留原始正负号，并将节点 `i` 的每条出边权值设为 `1 / d_i`，其中 `d_i` 是其出邻居数量，以表示节点把注意力平均分配给所有出邻居。实验参数包括：

- `xi = 2`
- 学习率 `lr = 0.01`
- 折扣因子 `gamma = 0.95`
- 时间步默认 `T = 30`
- Python 3.10
- PyTorch 2.2

### 3.2 算法验证

作者将 T-DQN 与 5 个动作策略本身，以及一个 Q-Learning 基线比较。Q-Learning 使用与 T-DQN 相同的候选集、奖励函数和动作空间，但把连续状态离散化成区间，用表格 Q 学习近似。

#### 3.2.1 不同种子数下的性能

固定 `T=30`，比较不同种子数量下的最终总体观点。结果显示，随着种子数增加，最终观点总和持续增大。T-DQN 整体优于 5 个启发式基线，并在 6 个网络中的 5 个上优于 Q-Learning。作者认为这是因为 DQN 更擅长处理连续状态，而 Q-Learning 的状态离散化会丢失细节。

在启发式策略中：

- `MaxDegree` 和 `CbC` 整体最弱；
- `Blocking` 在多数网络上表现较强；
- `MixStrategy` 在 Gnutella 这类稀疏网络上更有适应性。

#### 3.2.2 不同传播时间下的性能

固定种子数为 40，比较不同时间步下的表现。总体观点通常先明显上升，然后趋于平缓，甚至在部分情况下略有下降。T-DQN 整体优于启发式方法和 Q-Learning，尤其在传播时间足够长时优势更稳定。在稀疏网络 Gnutella 上，短时间内 T-DQN 的领先并不明显，但随着传播时间增加，优势逐渐体现。

#### 3.2.3 回音室现象下的性能

作者在 SBM 网络上额外模拟“回音室”现象：同一社区内节点初始观点更接近。具体做法是：

1. 为每个社区随机生成一个观点中心，范围在 `[-1,1]`；
2. 社区内节点初始观点从以该中心为均值、标准差 `0.2` 的高斯分布中采样；
3. 再裁剪到 `[-1,1]` 范围。

结果表明：

- 种子数增多时，总体观点上升；
- 时间步过长时，总体观点反而下降。

作者认为，在回音室结构下，短时间传播主要发生在社区内部，由于同社区观点相近且多数边为正边，容易得到较高局部收益并促进合作；但当传播跨越社区边界后，异质观点增多，会削弱收益和合作水平，从而降低最终总体观点。在这种设置下，T-DQN 仍然整体优于 5 个启发式算法，并且随种子数增加，优势更明显。

## 4. 仿真实验与分析

第 4 节主要研究两类因素对最终观点的影响：

1. **初始合作比例**
2. **网络结构**

作者通过仿真说明，网络中初始合作比例越高，往往越有利于观点传播和总体观点提升。因为更多节点愿意在早期进行观点交互，使正向观点更容易扩散并稳定下来。

另一方面，不同网络结构会显著影响观点最大化结果。稀疏网络、社区结构明显的网络和具有超级节点的网络，对种子选择策略的敏感性不同。例如：

- 在有明显超级节点的网络中，高入度节点可能天然占据主导作用；
- 在稀疏网络中，单纯依赖局部中心性不一定足够；
- 在社区结构清晰的网络中，跨社区传播能力非常关键。

这些实验表明，观点最大化算法不能只依赖单一中心性指标，而应根据网络结构动态选择策略，这也是 T-DQN 的主要优势来源。

## 5. 结论

本文围绕社会信任网络中的观点最大化问题，提出了两个核心创新：

1. 把个体固执性建模为合作/背叛的动态策略选择，构建了基于博弈论的连续观点动力学模型；
2. 提出两阶段 T-DQN 算法，用候选筛选缩小搜索空间，再用 DQN 在 5 种选种策略之间自适应决策。

实验表明，T-DQN 在多种网络、不同种子数、不同传播时长以及回音室场景下都表现出较强的有效性与鲁棒性。

论文也指出了后续可扩展方向，例如：

- 研究动态变化的社会网络；
- 考虑多主题观点传播；
- 进一步优化大规模网络上的计算效率。
